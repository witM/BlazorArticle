Absolute Value

Definition

The absolute value of a number \$a\$ is a number such that: \[ |a| = \begin{cases} a & \text{for} \ a \geq 0 \\ -a & \text{for} \ a < 0 \end{cases} \]

For a non-negative number, we simply drop the absolute value symbols ||, and for a negative value, we drop the symbols and change the sign of the number to the opposite. Therefore, for every \$a \in \mathbb{R}\$, we have \$|a| \geq 0\$.

[[[VideoItem Code="s_absolute_value_basics"]]]

We can interpret the absolute value as the distance of the number \$a\$ from zero on the number line. This interpretation also helps in solving certain equations and inequalities involving absolute value.

Several examples of absolute value.

\$|15| = 15\$

\$|-97| = 97\$

\$\displaystyle{\left|\frac{7}{5}\right| = \frac{7}{5}}\$

\$|-\sqrt{2}| = \sqrt{2}\$

Calculate the value of the expression \$|1-\sqrt{2}|\$ by removing the absolute value.

Using a calculator, we approximate \$\sqrt{2} \approx 1.4 > 1\$, so the expression \$1-\sqrt{2}< 0\$. We remove the absolute value by changing the sign.

\[\begin{aligned} |1-\sqrt{2}| &= -(1-\sqrt{2}) \\ &= -1+\sqrt{2} \\ &\approx 0.4 \end{aligned}\]

Formulas

The basic laws of absolute value operations for \$a,b \in \mathbb{R}\$.

\[ |a*b|=|a|*|b| \] \[ \left|\frac{a}{b}\right| = \frac{|a|}{|b|} \ \text{for} \ b\neq 0 \] \[ |a-b|=|b-a| \] \[ |a^2|=|a|^2 = a^2 \] \[ \sqrt{a^2} = |a| \]

\$|7-5|=|2|=|5-7| = |-2|\$

\$|39-15|=|24|=|15-39| = |-24|\$

\$|-8-(-3)|=|-5|=|-3-(-8)| = |5|\$

\$\sqrt{2^2}=\sqrt{4} = 2 = |2|\$

\$\sqrt{(-2)^2}=\sqrt{4} = 2 = |-2|\$

\$\sqrt{7^2}=\sqrt{49} = 7 = |7|\$

\$\sqrt{(-7)^2}=\sqrt{49} = 7 = |-7|\$

\$\sqrt{x^2}=x, \ \text{for} \ x\ge 0\$

\$\sqrt{x^2}=|x|, \ \text{for} \ x\in\mathbb{R} \$
